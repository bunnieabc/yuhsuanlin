Blender


Overview



Blender is Sketch Plugin which lets Sketch users blend colors, shape, opacity, and size between two shapes that share the same type. You can blend objects to create and distribute shapes evenly between multiple objects in Sketch. I designed and developed this plugin to speed up time for doing UI design. This project gains 297 stars on Github. Have fun and enjoy it!

Problems

Sketch is a powerful software for interface design. Some designers even use it to do simple graphics design. However, there is no blend tool in Sketch. When designers want to blend colors or shape, they still need to open Illustrator to do the trick. Furthermore, when designers want to create a large number of copies, they need to do “copy and paste” many times which waste a lot of time.


Interviews


Solution

To solve the problems, I designed and developed a sketch plugin which is similar to Illustrator’s blend tool. Through this Plugin, users can blend objects to create and distribute shapes evenly between multiple objects in Sketch. It is very easy to use. Users only need to download and double click to install the plugin.


Basic Usage


  • Create 2 or more shapes and put the layers in the right order. It will create and distribute shapes from the bottom layer to the top layer.
  • Select shapes that you want to blend.
  • You will see a input box. Type the number of steps you want between those shapes.
  • Then you get the result.
Note: The two shapes should share the same type.
